## App Icon Requirements

### Sizes Needed (iOS)
- **20x20** - Notification (2x, 3x)
- **29x29** - Settings (2x, 3x)
- **40x40** - Spotlight (2x, 3x)
- **60x60** - App (2x, 3x)
- **1024x1024** - App Store

### Design Tips
1. **Simple & Recognizable** - Should work at small sizes
2. **No Text** - Icons don't scale well with text
3. **Funko Pop Theme** - Consider a Pop figure silhouette or box
4. **Brand Colors** - Use your app's accent color
5. **Rounded Corners** - iOS automatically applies, design for square

### Design Ideas
- **Option 1:** Funko Pop box with barcode scanner overlay
- **Option 2:** Stylized Pop figure head
- **Option 3:** Collection bin/tray icon
- **Option 4:** Pop box with dollar sign (value tracking)

### Tools
- **Figma** - Free, web-based
- **Sketch** - Mac only
- **Adobe Illustrator** - Professional
- **Canva** - Easy, templates available

## App Store Screenshots

### Required Sizes
- **iPhone 6.7" Display** (iPhone 14 Pro Max): 1290 x 2796
- **iPhone 6.5" Display** (iPhone 11 Pro Max): 1242 x 2688
- **iPhone 5.5" Display** (iPhone 8 Plus): 1242 x 2208
- **iPad Pro 12.9"**: 2048 x 2732

### Screenshot Ideas
1. **Home Screen** - Total collection value, stats
2. **Scanning** - Barcode scanner in action
3. **Collection** - Organized bins with Pops
4. **Stats** - Beautiful charts and insights
5. **Wishlist** - Tracking wanted Pops
6. **Price Alerts** - Notification example

### Design Tips
- Show real content (not placeholders)
- Highlight key features
- Use device frames (optional)
- Add text overlays explaining features
- Keep it clean and uncluttered

## Quick Start

1. **Design Icon** (1024x1024)
   - Use design tool of choice
   - Export as PNG
   - Use online tool to generate all sizes

2. **Create Screenshots**
   - Run app in Simulator
   - Take screenshots of key screens
   - Add text/annotations if needed
   - Export in required sizes

3. **Add to Xcode**
   - Open Assets.xcassets
   - Add AppIcon set
   - Drag icons to appropriate slots
   - Add screenshots to App Store Connect
